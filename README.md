# Personal dotfiles
Symlinks managed with [GNU stow](https://www.gnu.org/software/stow/)

## Current setup
- OS: Arch Linux
- Shell: Fish
- DE: KDE Plasma
- WM: Wayland
- Terminal: Konsole
    - Startup command: `/usr/bin/fish -c "echo; pwd; ls; echo; fastfetch; exec fish"` 
- Font: [Monocraft](https://github.com/IdreesInc/Monocraft)
